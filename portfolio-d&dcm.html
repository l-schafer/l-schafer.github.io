D&D Character Manager

  • Category: Desktop Development (Java)
  • Course: Software Design Patterns
  • Completed: 10 November, 2021

Overview

This was our semester-long project for Software Design Patterns.

As part of a team of five, I was responsible for organization of the GUI and GUI backend. However, I did end up writing a fair portion of the backend as well. The design incorporates multiple software design patterns (as per requirement) which resulted in a complex, structurally-sound code structure.

While I have not played D&D myself, other members of the group and of the class seemed impressed with the considerations made and the intuitive interconnectivity between components.
